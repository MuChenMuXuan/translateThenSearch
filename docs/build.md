# 新用户首次注册chrome web store开发者

## 1、需要支付5美元，才能成为开发者。

我用的是国内银行的visa双币信用卡。

地区选择的是“香港”，这里的地区可以不是银行卡归属的地区。“地址”去网上搜一个香港真实存在的地址就可以。
